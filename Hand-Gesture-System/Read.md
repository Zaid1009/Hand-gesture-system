# Hand Gesture Recognition System

This project detects and classifies hand gestures using Python, OpenCV, and MediaPipe.  
It can recognize specific hand signs and perform actions based on them.

## Features
- Real-time hand tracking
- Gesture classification
- Easy to customize for different applications

## Technologies Used
- Python
- OpenCV
- MediaPipe

## How to Run
1. Clone the repo
2. Install dependencies (`pip install -r requirements.txt`)
3. Run the file (The gesture you want to run)
